# カテゴリ・著者・トップページのSEO設定

> Meta Description 関連エラーの解消用。SEO SIMPLE PACK で各ページにメタを設定する。

---

## 0. トップページ（`https://growfolio-note.com/`）

### 原因

現在のメタディスクリプションが **キャッチフレーズのみ**（「つくって、増やす、AI×投資の実践ノート」＝約20字）で、Bingの推奨（150字前後）を大きく下回っています。

### 設定手順

1. **SEO PACK** → **ホームページ**（または **トップページ**）タブを開く
2. **メタディスクリプション**（または「ディスクリプション」）欄に下記を貼り付け
3. 必要なら **SEOタイトル** も確認（現状のままでOK）
4. **保存**

> 💡 欄が見つからない場合：**SEO PACK → 一般** で「ホームページのメタ情報を個別に設定」が有効か確認。
> 💡 **設定 → 一般 → キャッチフレーズ** はサイト内表示用の短い文言のままでOK。メタはSEO PACK側で上書きします。

### コピペ用

**SEOタイトル（現状維持でOK）**
> グロウフォリオ | つくって、増やす、AI×投資の実践ノート

**メタディスクリプション（148字）**
> グロウフォリオは、AI開発と仮想通貨・資産運用を実際に手を動かしながら検証し、その過程と結果を記録する個人ブログです。BitradeXの始め方・KYC・入金・AIプラン比較・出金方法・リスク・紹介報酬・確定申告まで、画像付きの実践ガイドと正直なレビュー記事を発信しています。※投資は自己責任です。

### 設定後の確認

ページのソースに以下が出ていればOK：
```html
<meta name="description" content="グロウフォリオは、AI開発と仮想通貨・資産運用を...">
```

## 1. カテゴリページ

1. WordPress管理画面 → **投稿** → **カテゴリー**
2. 対象カテゴリの **編集** をクリック
3. ページ下部の **SEO SIMPLE PACK** 欄（または「SEO設定」）を開く
4. 下表の **メタディスクリプション** を貼り付け
5. 必要なら **SEOタイトル** も設定（未設定ならカテゴリ名＋サイト名で自動生成される）
6. **更新** をクリック

> 💡 SEO SIMPLE PACK の欄がカテゴリ編集画面に出ない場合：**SEO PACK** → **その他** → **カテゴリページのメタ情報を個別設定** を有効化。

### 著者アーカイブページ

> ⚠️ カテゴリと違い、**著者ごとの個別設定欄はありません**。
> **SEO PACK → 「著者」のアーカイブページ** の「ディスクリプションの形式」で、全著者共通のテンプレートを設定します。

1. **SEO PACK** → **「著者」のアーカイブページ** タブを開く（今見ている画面）
2. **ディスクリプションの形式** に下記を貼り付け
3. 必要なら **タイトルタグの形式** も変更
4. **保存**

**タイトルタグの形式（任意）**
```
%_author_name_%の記事一覧 %_sep_% %_site_title_%
```
プレビュー例：`owasoftの記事一覧 | グロウフォリオ`

**ディスクリプションの形式（ここに貼り付け）**
```
%_author_name_%の記事一覧。グロウフォリオはAI開発と仮想通貨・資産運用の実践記録を発信する個人ブログです。実体験に基づく再現性のある情報をお届けします。
```
プレビュー例：`owasoftの記事一覧。グロウフォリオはAI開発と仮想通貨・資産運用の実践記録を発信する個人ブログです。実体験に基づく再現性のある情報をお届けします。`

> 💡 `%_author_name_%` は著者の表示名（owasoft）に自動置換されます。

#### noindex について（重要）

スクリーンショットの **「著者」のアーカイブページをインデックスさせない」が「はい」** になっています。

| 設定 | おすすめ | 理由 |
| --- | --- | --- |
| インデックスさせない：**はい**（現状） | 個人ブログで著者が1人なら **このままでOK** | 著者ページは記事一覧の重複になりやすく、SEO上はnoindexが一般的 |
| インデックスさせない：**いいえ** | 著者ページを検索に出したい場合のみ | Bingのエラー対象にしたくない場合 |

noindex でも meta description タグ自体は出力されるので、**Meta Description エラー解消には「ディスクリプションの形式」の変更だけで十分**です。

## 設定後の確認

1. 各URLをブラウザで開き、**ページのソースを表示**
2. `<meta name="description" content="...">` が存在するか確認
3. Bing Webmaster Tools / サーチコンソールで再クロールをリクエスト

### 確認コマンド（任意）

```powershell
$urls = @(
  "https://growfolio-note.com/category/start/",
  "https://growfolio-note.com/category/risk/",
  "https://growfolio-note.com/category/performance/",
  "https://growfolio-note.com/category/withdrawal/",
  "https://growfolio-note.com/category/referral/",
  "https://growfolio-note.com/category/tax/",
  "https://growfolio-note.com/author/owaso30/"
)
foreach ($url in $urls) {
  $html = curl -sL $url
  $has = if ($html -match 'meta name="description"') { "OK" } else { "NG" }
  Write-Output "$has : $url"
}
```

すべて `OK` になれば Meta Description エラーは解消されます。

## 2. 固定ページ（タイトルが短すぎるエラー解消）

### 原因

Bingが指摘した3ページは、タイトルが **「ページ名 | グロウフォリオ」**（約13字）と短すぎます。推奨は **50字前後**。

| ページ | 現状のタイトル（短い） | 文字数 |
| --- | --- | --- |
| `/profile/` | 運営者情報 \| グロウフォリオ | 約14字 |
| `/contact/` | お問い合わせ｜グロウフォリオ（…） | 約31字 |
| `/disclaimer/` | 免責事項 \| グロウフォリオ | 約13字 |

### 設定手順

1. **固定ページ** → 対象ページの **編集**
2. ページ下部の **SEO SIMPLE PACK** → **SEOタイトル** を変更
3. **更新**

### コピペ用 SEOタイトル（50字前後）

**運営者情報（`profile`）— 52字**
```
運営者情報・プロフィール｜グロウフォリオ（owasoft）｜AI開発と仮想通貨・資産運用の実践記録ブログ
```

**お問い合わせ（`contact`）— 49字**
```
お問い合わせ｜グロウフォリオへのご連絡・ご質問窓口｜記事の誤記報告・取材依頼・広告掲載のご相談受付
```

**免責事項（`disclaimer`）— 49字**
```
免責事項｜投資・仮想通貨・BitradeXに関する免責・注意事項とPR表記について｜グロウフォリオ
```

**プライバシーポリシー（`privacy-policy`）— 予防的に設定推奨**
```
プライバシーポリシー｜個人情報の取り扱いとCookieの利用について｜グロウフォリオ
```

### メタディスクリプションも未設定 → ついでに設定推奨

現状、以下3ページは **meta description タグも出力されていません**。Bingで次のエラーになる前に、SEO SIMPLE PACK の **メタディスクリプション** 欄にも設定しておくと安心です（SEOタイトルと同じ編集画面）。

**運営者情報（`profile`）**
```
グロウフォリオの運営者owasoftのプロフィール。AI開発×仮想通貨×資産形成の実践記録ブログの発信方針・運営者について・免責へのリンクを掲載しています。
```

**免責事項（`disclaimer`）**
```
グロウフォリオの免責事項。投資・仮想通貨に関する免責、金融商品取引法上の表記、アフィリエイト・広告（PR）表記、著作権について掲載しています。
```

**プライバシーポリシー（`privacy-policy`）**
```
グロウフォリオのプライバシーポリシー。お問い合わせ時の個人情報の利用目的、Googleアナリティクス（GA4）・広告・アフィリエイトにおけるCookieの利用について説明しています。
```

> 固定ページは検索流入の主力ではないが、**E-E-A-T（信頼性）ページ**なのでメタ未設定のままより設定済みの方がよい。お問い合わせページと同じ要領で、**SEOタイトル＋メタディスクリプションを同時に**入れて更新するのが効率的。
