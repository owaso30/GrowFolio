## このブログについて

「**グロウフォリオ**」は、AI開発と資産運用を**実際に手を動かしながら検証し**、その過程と結果を記録する個人ブログです。

「つくって、増やす。」——自分でツールを作り、自分で運用し、そのリアルをそのまま書き残す。それがこのブログのコンセプトです。

掲載している情報は、ネット上の二次情報をまとめたものではなく、**筆者が実際に試したうえでの記録・所感**を中心に構成しています。うまくいったことも、失敗したことも、等身大で書くことを心がけています。

### 主な発信テーマ

- **AI開発・生産性** — エディタ、音声入力、自動化など、開発現場で使えるツールの実践
- **仮想通貨・資産運用** — BitradeX など、実際に運用しているサービスの手順・リスク・記録
- **ITキャリア** — Cursor、GitHub Copilot、VS Code など、開発者向けツールの比較と活用法

### バックグラウンド

owasoft は、ソフトウェア開発を軸に、AI・自動化・資産運用の領域を横断して活動している個人です。

日常の開発作業では、タイピングの負担を減らす音声入力や、AIを活用した文章・コード生成を積極的に取り入れています。ブログ運営についても、記事生成・画像作成・公開スケジュールを GitHub Actions で自動化し、「作る仕組み」と「書く内容」の両方を自分で設計・運用しています。

投資・資産運用については、特定の金融商品を推奨する立場ではなく、**自分の資金で実際に運用した記録**を公開しています。BitradeX については、口座開設から入金・AIプラン運用・出金まで一通り経験しており、記事内では可能な範囲でスクリーンショットや残高の推移も掲載しています。

### 開発プロダクト：Voxera

2026年、Windows 向けの音声入力アプリ「**[Voxera](https://voxera-voiceapp.com/)**」をリリースしました。

Voxera は、ホットキーで話すだけでカーソル位置に文字が入力されるデスクトップ向けの音声入力ツールです。Whisper ベースの高精度な文字起こしに加え、**多言語ハブ**機能を搭載しており、日本語で話した内容を英語のメールや Slack 向けの文体に整えて、そのまま入力欄へ流し込むことができます。

開発のきっかけは、自分自身がグローバルなコミュニケーションや長文の下書きで「タイピングと翻訳タブの往復」に時間を取られていた課題への応答です。エンジニア・PM・ライターなど、日々テキストを扱う人の生産性を上げることを目的に設計しています。

> Voxera の詳細・ダウンロードは [voxera-voiceapp.com](https://voxera-voiceapp.com/) をご覧ください。

## 発信方針

グロウフォリオでは、以下の方針で記事を書いています。

1. **実体験ベース** — 自分で試していないことは、あたかも経験したかのように書かない
2. **リスクの明示** — 投資・仮想通貨に関する記事では、元本割れ・詐欺リスク・規制上の注意点を必ず記載する
3. **PR・アフィリエイトの開示** — 紹介リンクやアフィリエイトを含む記事には、記事内に PR 表記を行う
4. **情報の更新** — サービスの仕様変更や画面の変化があれば、読者からの指摘や自身の確認をもとに記事を修正する
5. **再現性の重視** — 手順記事はスクリーンショット付きで、読者が同じ操作を追えることを優先する

## このブログで書かないこと

以下については、ブログの性質上、お答えできない・書かない領域です。

- 個別の投資判断（「今買うべきか」「いくら入れるべきか」など）
- 税務・法律に関する個別の助言（一般的な情報の紹介にとどめます）
- 各サービスの公式サポートに代わる口座・入出金の代行

投資・税務に関する最終判断は、必ずご自身の責任で行い、必要に応じて専門家にご相談ください。
