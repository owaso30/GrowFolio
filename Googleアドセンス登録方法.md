# Google AdSense 登録方法（グロウフォリオ / WordPress × SWELL）

> 目的：グロウフォリオ（`https://growfolio-note.com`）で Google AdSense の審査申請〜広告設置までを、迷わず完了できるように手順をまとめる。  
> 対象環境：エックスサーバー × WordPress × SWELL × SEO SIMPLE PACK  
> 所要：事前準備が済んでいれば 30〜60 分。審査結果は数日〜最大 4 週間。

---

## 0. 始める前に知っておくこと


| 項目     | 内容                                                      |
| ------ | ------------------------------------------------------- |
| 公式入口   | [Google AdSense](https://www.google.com/adsense/start/) |
| アカウント数 | **1 人 1 アカウント**（複数作成・譲渡は規約違反）                           |
| 初回審査   | **独自ドメイン必須**（`growfolio-note.com` のようなルートドメインで OK）      |
| 審査期間   | 通常 数日〜2 週間、長い場合 **最大 4 週間**                             |
| 収益化の流れ | 申請 → 審査 → 承認 → 広告コード設置 → 収益発生 → 閾値到達後に振込                |


> ⚠️ **Site Kit（Google 公式プラグイン）は入れない**  
> GA4 は SEO SIMPLE PACK 経由で設定済み。Site Kit を追加すると二重計測・プラグイン競合の原因になる（[WordPress初期セットアップ手順書](BitradeX/WordPress初期セットアップ手順書.md) §6-1 参照）。

## 1. 申請前チェックリスト

AdSense 審査は「サイト全体の品質」を見る。グロウフォリオの現状目安は **公開記事 13 本以上 + 固定ページ 4 種** で申請可能圏内（[automation/README.md](automation/README.md) 参照）。

### 1-1. 必須の固定ページ

- [x] **運営者情報（プロフィール）** — 実運用者としての経歴・発信方針
- [x] **プライバシーポリシー** — Cookie / GA4 / アフィリエイト / 広告（AdSense）利用の明記
- [x] **免責事項** — 金融 YMYL 向け（元本非保証・投資は自己責任・PR 表記）
- [x] **お問い合わせ** — Contact Form 7 等で送信可能なフォーム

> フッターメニューから各ページへリンクできる状態にする（SWELL：外観 → メニュー → フッター）。



### 1-2. コンテンツ・技術面

- [x] 公開記事が **10〜20 本程度** ある（各記事に独自性・情報量があること）
- [x] **HTTPS** で全ページが正常表示される
- [x] **モバイル** で読みやすい（SWELL 標準で概ね OK）
- [x] **ナビゲーション** が分かりやすい（カテゴリメニュー + フッター固定ページ）
- [x] **404・死リンク** が目立たない
- [x] **Google Search Console** 登録済み・サイトマップ送信済み（`/sitemap.xml`）
- [x] 記事に **免責・PR 表記** がある（アフィリエイト併用サイトは特に重要）



### 1-3. 審査で不利になりやすい点（グロウフォリオ向け）


| 注意点              | 対策                             |
| ---------------- | ------------------------------ |
| 金融・投資・仮想通貨（YMYL） | 事実と考察の分離、複数ソース、免責の明記を徹底        |
| アフィリエイト過多        | 本文は情報価値優先。広告・アフィリ CTA は適度に     |
| 記事の薄さ・重複         | 同 KW のカニバリを避ける。1 日複数本の一括公開は避ける |
| コピーコンテンツ         | LLM 生成でも事実確認・独自の整理・考察を入れる      |

## 2. AdSense アカウント開設



### STEP 1. Google アカウントでログイン

- ブログ運用用 Gmail（Search Console / GA4 と **同じ Google アカウント** 推奨）で [AdSense 公式](https://www.google.com/adsense/start/) にアクセス
- 「**利用を開始**」または「**今すぐ登録**」をクリック



### STEP 2. サイト情報を入力


| 項目              | 入力例（グロウフォリオ）                               |
| --------------- | ------------------------------------------ |
| あなたのサイト         | `growfolio-note.com`（`https://` 以降のドメイン部分） |
| AdSense からの情報配信 | どちらでも可                                     |
| お支払先の国または地域     | **日本**                                     |
| 利用規約            | 内容を確認し **同意**                              |


→ 「**AdSense の利用を開始**」で次へ

### STEP 3. お客様情報（支払い情報）を入力


| 項目       | 内容                    |
| -------- | --------------------- |
| アカウントの種類 | 個人ブログなら **個人**        |
| 名前と住所    | **本人確認書類と一致** する正確な情報 |
| 電話番号     | SMS または音声での確認に使用      |


> 住所・氏名の誤りは後の振込で問題になる。**ローマ字表記の要件** がある場合は AdSense 画面の指示に従う。



### STEP 4. 電話番号認証（求められた場合）

- 表示された認証コードを SMS または音声通話で受け取り、入力して完了

---



## 3. 審査用コードの設置（SWELL）



### STEP 1. AdSense 側でコードを取得

1. AdSense 管理画面 →「**サイト**」→ 対象サイトの「**サイトを AdSense にリンク**」→「**開始**」
2. 確認方法：**AdSense コード スニペット** を選択
3. 表示された `<script async src="https://pagead2.googlesyndication.com/...">` 形式のコードを **コピー**



### STEP 2. SWELL にコードを貼り付け

1. WordPress 管理画面 →「**外観**」→「**カスタマイズ**」

2.「**高度な設定**」を開く
3.「**head タグ終了直前に出力するコード** `</head>` 直前」欄に、コピーした AdSense コードを貼り付け
4. 上部の「**公開**」をクリックして保存

> 子テーマ **SWELL CHILD** が有効な状態で作業する（親テーマ SWELL を直接編集しない）。



### STEP 3. AdSense 側で配置確認

1. AdSense の画面に戻る

2.「**コードを配置しました**」にチェック →「**確認**」
3.「サイトが確認されました」と表示されたら「**次へ**」
4.「**審査をリクエスト**」をクリック

### STEP 4. サイトの同意メッセージ（CMP）— 表示された場合

「**サイトの同意メッセージを作成する**」画面が出たら、**Google 認定 CMP** をその場で有効化する手順です。EEA（欧州）・英国・スイスからのアクセス向けに、Cookie / 広告の同意を取るためのバナーが自動で設置されます。

| 選択肢 | 内容 | おすすめ |
|--------|------|----------|
| **左（2 択）** | 「同意する」「オプションを管理する」 | シンプル。日本向けメインなら十分なことが多い |
| **中央（3 択）** | 「同意する」「同意しない」「オプションを管理する」 | **推奨**。拒否ボタンが最初から見える分、欧州向け規制に沿いやすい |
| **右（代替 CMP）** | Cookiebot 等の第三者 CMP を別途導入 | 特別な理由がなければ **選ばない** |

**グロウフォリオでの推奨操作**

1. **中央の「3 択」** を選択（迷ったらこちら）
2. 右下の「**送信**」をクリック
3. Google が同意バナー用コードをサイトに反映（審査コードと同様、head 周辺に追加される想定）

> 「**後で通知**」でも審査自体は進むが、欧州圏ユーザー向け広告収益を失うリスクがある。**今のうちに 3 択 + 送信** で済ませるのが無難。  
> 第三者 CMP は設定・費用・SWELL との相性確認が増えるため、個人ブログでは Google 標準 CMP で足りる。

これで審査申請は完了。結果は AdSense 管理画面と登録メールで通知される。

## 4. 審査中・審査後の運用



### 審査中にやること

- サイトを **公開状態** のまま維持する
- **大規模リニューアル** より、小さな更新を継続
- 固定ページ・免責・プライバシーポリシーのリンク切れがないか確認
- 1 日に複数記事を一括公開しない（審査・クロール品質に不利）



### 審査合格後

1. AdSense から「**アカウントが有効になりました**」系のメールが届く
2. 管理画面 →「**広告**」→「**広告ユニット**」または「**自動広告**」で広告を作成
3. SWELL への設置場所（審査用コードとは別）：


| 広告の種類  | SWELL での設定場所                              |
| ------ | ----------------------------------------- |
| 自動広告   | SWELL設定 → 広告コード →「自動広告 [ad]」              |
| 記事内広告  | SWELL設定 → 広告コード →「記事内広告 [ad]」＋ 本文に `[ad]` |
| 目次前後   | SWELL設定 → 広告コード →「目次広告 [ad]」              |
| サイドバー等 | 外観 → ウィジェット →「[SWELL] 広告コード」              |


> 審査用コード（head 内）は、合格後も残しておいて問題ない。自動広告を有効にした場合は AdSense 側の設定と SWELL 側の `[ad]` タグの **二重設置** に注意。



### プライバシーポリシーの追記（合格後）

AdSense 利用開始後、プライバシーポリシーに以下を追記する。

- Google AdSense による Cookie 利用
- 第三者配信事業者による広告配信
- パーソナライズド広告のオプトアウト方法（[Google 広告設定](https://www.google.com/settings/ads) 等）

EU 向けトラフィックがある場合は、AdSense 管理画面の **同意メッセージ（CMP）** 設定も求められることがある。

## 5. 振込（支払い）設定

審査合格後、収益が発生しても **支払い情報が未完了** だと振込されない。

1. AdSense →「**お支払い**」→ 支払いプロファイルを完成させる
2. **本人確認**（住所確認ハガキ等）を完了
3. **振込先口座**（国内銀行）を登録
4. **支払いのしきい値**（例：8,000 円）に達した月の翌月以降に振込

> 税務上、AdSense 収益は **雑所得等** として確定申告の対象になり得る。必要に応じて税理士・国税庁情報を参照。

## 6. 審査に落ちた場合

Google は **具体的な不合格理由を必ずしも開示しない**。通知文と AdSense ヘルプの該当項目を手がかりに修正する。


| よくある理由      | 対処                           |
| ----------- | ---------------------------- |
| 有用性の低いコンテンツ | 記事を厚くする・独自性を足す・重複記事を統合       |
| 必須ページ不足     | 運営者情報・プライバシー・お問い合わせを整備       |
| ナビゲーション不足   | メニュー・フッターリンク・カテゴリ設計を見直し      |
| ポリシー違反疑い    | 誇大表現・コピーコンテンツ・禁止コンテンツを除去     |
| サイトが確認できない  | SSL・DNS・メンテナンス表示・Basic 認証を確認 |


**再申請の目安**：修正後 **1〜2 週間以上** 間隔を空け、品質を上げてから再度「審査をリクエスト」。焦って何度も申請すると不利になりやすい。

## 7. グロウフォリオ向け・申請前最終確認

申請ボタンを押す直前に、以下をブラウザで確認する。

- [ ] `https://growfolio-note.com/` が HTTPS で表示される
- [ ] フッターから **運営者情報 / プライバシー / 免責 / お問い合わせ** に遷移できる
- [ ] 代表記事を 2〜3 本開き、免責・PR・読みやすさに問題がない
- [ ] `https://growfolio-note.com/sitemap.xml` が XML として開ける
- [ ] Search Console にエラーが大量に出ていない
- [ ] ページソース（表示 → ページのソース）で AdSense 審査コードが `</head>` 直前にある

## 9. 作業チェックリスト（印刷用）

```

□ お客様情報・電話認証

```
